ספר ומדריך php 

הספר php הזה ילווה אותך בתהליך לימוד php וייתן לך הסברים פשוטים על שפת php, המבנה שלה ואפשרויות שלה. לקראת סוף הספר אתה תהיה מסוגל בקלות לבנות אתר ולכתוב ולהבין קוד php.

אתה יכול להעתיק, להדפיס, לצלם או לשלוח את המדריך php הזה לכל החברים או אפילו לערוך אותו בעצמך אם יש לך את הידע המתאים. הספר php נכתב (בלשון הווה) על ידי קהילת מפתחי php הישראלית, אנשים בעלי נסיון שלמדו ועובדים עם php מזה כמה שנים.
